Website Planning Document

Site Name: Mayan Legacy

The name "Mayan Legacy" was chosen because it reflects both the historical significance and the enduring cultural impact of this ancient civilization, which continues to fascinate historians and travelers alike.

Optional domain availability: mayanlegacy.org

Site Purpose

This site is intended to provide relevant information about the Mayan Culture and Legacy and the most recent discoveries and news related to this amazing civilization.

Scenarios

This website is mainly directed to history enthusiasts but it is also directed to tourists planning to visit archeological sites in México.

This site answers these two kinds of questions:

  1. What are the most recent archaeological discoveries about the Mayan civilization?
  2. What are the main Mayan archaeological sites I can visit and how do I get there?

Color Schema

The color palette is inspired by the Bonampak murals — the best-preserved Maya murals in existence. Each color is taken directly from the pigments and imagery found in this ancient artwork.

Color Hex Use
Pacific Blue #2bb8c6 Header, navigation, hero section background, card borders
Rusty Spice #be2e04 H1, H2 headings, call-to-action buttons, highlighted accents
Sea Green #00884b H3 headings, links, icons, decorative details
Antique White #f4e5d1 Page background, card backgrounds
Rich Mahogany #4e1009 Body text paragraphs, footer text
